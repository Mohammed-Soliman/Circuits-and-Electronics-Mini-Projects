# Circuits-and-Electronics-Mini-Projects

This repository contains a collection of foundational **analog and digital electronics projects**, implemented using discrete components and basic ICs. The projects are designed to reinforce core concepts in circuit analysis, power systems, and electronic design.

Course: CIE 110 – Circuits and Electronics  
Institution: Zewail City of Science and Technology  
Semester: Spring 2025  

---

## 🚀 Project Overview

The project consists of five hands-on circuit implementations:

---

### 1️⃣ Diode-Based Logic Gate Circuits
- Implementation of basic logic gates using diodes and resistors
- Gates implemented: **OR, AND, NOR, NAND**
- LED indicators for output visualization
- Breadboard-based design

📌 Concepts Demonstrated:
- Digital logic fundamentals  
- Diode switching behavior  
- Truth tables and logic design  
- Hardware implementation of Boolean functions  

---

### 2️⃣ Multi-Output Power Supply
- Converts **220V AC → regulated DC outputs**
- Provides **+12V, +5V outputs**
- Uses:
  - Transformer (step-down)
  - Bridge rectifier
  - Filtering capacitors
  - Linear voltage regulators (7805, 7809, 7812)
- LED indicators for each output

📌 Concepts Demonstrated:
- AC to DC conversion  
- Rectification and filtering  
- Voltage regulation  
- Power distribution for multiple loads  

---

### 3️⃣ Voltage Regulator with Zener Diode
- Converts AC input to **stable ~5V DC output**
- Uses:
  - Bridge rectifier
  - Smoothing capacitor
  - Zener diode (voltage stabilization)
  - Current-limiting resistor

📌 Concepts Demonstrated:
- Zener diode operation  
- Voltage clamping and regulation  
- Load stability  
- Basic power supply design  

---

### 4️⃣ Traffic Light System
- Simulates a real traffic light sequence
- Built using:
  - **555 Timer (clock generator)**
  - **CD4017 Decade Counter**
- Sequential LED control (Red → Yellow → Green)
- Adjustable timing using potentiometer

📌 Concepts Demonstrated:
- Sequential logic systems  
- Clock generation (astable multivibrator)  
- Counters and timing circuits  
- Real-world embedded system simulation  

---

### 5️⃣ Common Emitter Audio Amplifier
- Audio amplification using **2N2222 NPN transistor**
- Input: low-amplitude audio signal (e.g., phone output)
- Output: amplified signal driving a speaker
- Includes:
  - Biasing network
  - Coupling capacitors
  - Bypass capacitor for gain improvement

📌 Concepts Demonstrated:
- Transistor operation (BJT)  
- Common-emitter amplifier design  
- Signal amplification  
- AC/DC signal coupling and filtering  

---

## 🛠 Key Components Used
- Diodes (1N400x series, Zener diodes)  
- BJTs (2N2222)  
- ICs (555 Timer, CD4017, 4026)  
- Voltage regulators (7805, 7809, 7812)  
- Resistors, capacitors, potentiometers  
- Transformers and power adapters  
- LEDs, seven-segment displays, speakers
